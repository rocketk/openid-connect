      • isJwkAvailable

        public boolean isJwkAvailable()

        Checks if the JWK is already set for the processor. The value is not set if the jwk(JsonWebKey) is not called or the jwks(JsonWebKeySet) is not called or does not contain the key specified by the kid.

        If the algorithm is none then it will also return true as a JWK is not needed.

        Returns:
        jwk is set.

      • signatureCheck

        public JsonWebTokenProcessor signatureCheck(boolean b)
        Enable or disable signature checks. Signature checks are enabled by default. Signature checks can be disabled for scenarios where the signature had already been validated previously in the process and the JWT was stored as is rather than having the extracted payload kept.
