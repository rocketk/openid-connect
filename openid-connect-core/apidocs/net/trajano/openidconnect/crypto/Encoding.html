        public static String base64EncodeUint(BigInteger v)
        Encodes Base64urlUInt. The representation of a positive or zero integer value as the base64url encoding of the value's unsigned big endian representation as an octet sequence. The octet sequence MUST utilize the minimum number of octets needed to represent the value. Zero is represented as BASE64URL(single zero-valued octet), which is "AA".
