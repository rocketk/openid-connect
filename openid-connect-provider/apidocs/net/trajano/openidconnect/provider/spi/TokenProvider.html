net.trajano.openidconnect.provider.spi

Interface TokenProvider

  • All Known Implementing Classes:
    DefaultTokenProvider

    public interface TokenProvider
    This provides storage and retrieval for the token responses. Implementers would generally be providing an expiring cache that provides multiple key types pointing to the same token response instance.
